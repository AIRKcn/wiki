# SLA

> UGC：User Generated Content 的缩写，意思是用户原创内容。
> 互联网术语，全称为 User Generated Content，也就是用户生成内容的意思。UGC 的概念最早起源于互联网领域，即用户将自己原创的内容通过互联网平台进行展示或者提供给其他用户。UGC 是伴随着以提倡个性化为主要特点的 Web2.0 概念兴起的。

- 英文名 : User Generated Content
- 中文名 : 用户原创内容
- 特点 : 以提倡个性化为主
